# Pertinax Chess
This is a Chess game, that has an UCI compatible engine made by myself.

## Engine features 
NegaScout with alpha-beta pruning, TT.
Magic Bitboards, generally tabled attacks.

## Run the program

Requirements:
OpenGL 4.0 (or higher).
Run in an x64 windows machine.

Run program (found in the x64 directory), you'll only have the terminal window,
write "uci" then "setoption name Gui" to show the Gui.
On terminal type 'help' for help.
